Инструкция по загрузке изображений

Два способа загрузки: отдельные снимки или ZIP‑архив. Пошаговые инструкции с иллюстрациями ниже.

Одиночные изображения

1
Настройте параметры сжатия(ДО ЗАГРУЗКИ) и перетащите или выберите файл

Нажмите на область загрузки или перетащите изображение прямо в пунктирную зону. Поддерживаются PNG, JPG, JPEG, BMP, TIFF.

2
Расставление точек

Сначала активируйте режим, затем добавляйте положительные (зелёные) и отрицательные (красные) точки на холсте. Отрицательные точки исключают те области, которые не надо выделять(не обязательно расставлять, нужны в основном для дальнейшего уточнения области). Для повторной генерации нажмиите снова "Создать маску"

Положительные / отрицательные точки
3
Генерация маски

Нажмите "Создать маску" для предосмотра, на этом моменте изменения еще можно отменить. После нажатия "Заверишть маску", она попадает в итог и больше не редактируется. Расчет площади изначально производится с дефолтными параметрами, изменить можно как до загрузки изображений, так и после, данные обновятся

ZIP архив

1
Подготовьте архив

Так же, перед загрузкой архива необходимо настроить параметры сжатия. В ZIP должны быть только изображения (PNG, JPG, JPEG, BMP, TIFF). Желательно без вложенных папок.

2
Расставьте точки

Повторите такие же действия как с одиночной загрузкой. После расставления точек, нажмите "Создать маску", если необходимо создать только для одного изображения из архива, нажмите "Завершить маску", если для всех, нажмите "Распространить". Расчет площади изначально использует дефолтные значения, изменить можно как до загрузки, так и после загрузки изображений или архива

Архив загружен
3
Навигация

Для переключения между изображениями архива используется либо таблица, либо кнопки переключения

Маска на все изображения
Совет: Для массовой обработки используйте ZIP. После распространения можно скачать все маски одним архивом через кнопку «Скачать все» в панели результатов.

Поддерживаемые форматы

PNG, JPEG, JPG, BMP, TIFF.

Примеры изображений

Используйте вкладку «ZIP архив» → «Примеры изображений», чтобы протестировать без своих файлов.
