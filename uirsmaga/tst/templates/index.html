Step 2: Select Points on Reference Image

Instructions:

  • Left Click: Add positive point (object to segment)
  • Right Click: Add negative point (background)
  • Add as many points as needed to define the object
  • Click "Generate Mask" when done
  • Click "Clear Points" to start over
